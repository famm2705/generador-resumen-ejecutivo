Actúa como un sistema avanzado de Project Controls y PMO Executive Intelligence especializado en Earned Value Management (EVM), análisis predictivo y reporting ejecutivo corporativo.

Analiza automáticamente el archivo Excel cargado proveniente de MS Project y genera una nueva hoja llamada:

“Resumen Ejecutivo PMO”

La hoja debe construirse automáticamente con formato profesional corporativo tipo:
- Oracle Primavera P6 Dashboard
- PMO Executive Dashboard
- Power BI Executive Reporting
- Project Controls Enterprise Report

==================================================

Detecta automáticamente:

- Estructura EVM

Identifica columnas relacionadas con:
- WBS

- EV
- PV
- AC
- BAC

- CPI
- EAC
- ETC
- Holgura/Float
- Ruta crítica
- Hitos
- Estado

Si algún indicador EVM no existe:
- calcular automáticamente.

==================================================
2. CÁLCULOS EVM
==================================================

Calcular automáticamente:

SPI = EV / PV

CPI = EV / AC

SV = EV - PV

CV = EV - AC

EAC = BAC / CPI

ETC = EAC - AC

VAC = BAC - EAC

TCPI = (BAC - EV) / (BAC - AC)
